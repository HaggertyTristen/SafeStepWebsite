Technical Approach

Model Architecture

SafeStep utilizes AlexNet, a convolutional neural network (CNN) designed to classify and detect patterns within image data. The model processes aerial or ground-level images, distinguishing between “safe” and “mine” zones.

  • Input: 224×224 color images
  • Layers: Convolution → ReLU → Pooling → Fully Connected
  • Output: Binary classification (Mine / Safe)

Training Process

Training data was split into 80 training images and 6 validation images, organized in separate class folders. Data augmentation, normalization, and resizing ensured balanced learning. The model was trained for five epochs using the Adam optimizer, categorical cross-entropy loss, and learning rate tuning for optimal accuracy.
