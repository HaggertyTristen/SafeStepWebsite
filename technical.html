Technical Approach: AI + AlexNet

SafeStep System Overview

SafeStep is a wearable AI system integrated into boots and ankle devices. Using computer vision and deep learning, it identifies landmines before an operator steps on them.

  • Computer Vision + Deep Learning: Real-time mine pattern recognition
  • Instant Alerts: Vibration and visual signals for immediate feedback
  • Wearable Form Factor: Lightweight, rugged, and portable for field use

AlexNet Neural Engine

  • 8-Layer CNN: 5 convolutional + 3 fully connected layers
  • Feature Extraction: Detects subtle terrain patterns
  • Custom Training: Classifies "safe" vs "mine"
  • Real-Time Detection Pipeline: Boot camera → AlexNet → Confidence → Alert

Model Training & Validation

Models are trained in PyTorch with a combination of real and simulated minefield images.

  • Validation Accuracy: ~0.92
  • Training Accuracy: ~0.95
  • Validation Loss: ~0.18
  • False Positive Rate: ~0.05
  • False Negative Rate: ~0.03

Continuous improvement ensures minimal false alarms while maximizing safety and detection speed.

Roadmap & Scaling

  • Sensor Fusion: Add infrared, metal detection, and GPR layers
  • Dataset Expansion: Diverse terrains, lighting, climates
  • Edge Computing Optimization: Faster on-field response
  • Civilian Applications: Humanitarian demining, disaster relief, rescue operations
