Technical Approach

AlexNet Neural Engine

SafeStep uses an 8-layer convolutional neural network (5 convolutional + 3 fully connected layers) to classify terrain as "Safe" or "Mine." This network excels at detecting subtle terrain patterns from boot-mounted camera imagery in real time.

Real-Time Detection Pipeline

  1. Boot-mounted camera captures terrain images
  2. AlexNet analyzes each frame
  3. Confidence threshold triggers haptic & visual alerts
  4. Edge computing device (e.g., Jetson Nano) processes data in real time
