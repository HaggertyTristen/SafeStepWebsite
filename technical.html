Technical Approach

Using AlexNet for Landmine Detection

Architecture Overview

SafeStep uses a modified version of AlexNet, a deep convolutional neural network (CNN) originally designed for large-scale image recognition. We fine-tuned AlexNet to classify terrain images as Safe or Mine.

Process Summary

  • Input images from camera sensors (224x224)
  • Convolution + ReLU activation layers extract visual features
  • Pooling layers reduce noise and focus on key patterns
  • Fully connected layers classify terrain safety
  • Output: Probability of mine presence beneath the next step

Training Setup

Model trained in PyTorch on labeled data from Google Drive with “safe” and “mine” categories. Optimized with Adam optimizer, learning rate = 0.001, batch size = 16, for 5 epochs.
