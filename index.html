Technical Approach

We used a modified AlexNet convolutional neural network for classification. The network accepts 224x224 RGB images, processes them through five convolutional layers, and classifies outputs as Safe or Wet.

Training Process

  • Preprocessing: resized, normalized images
  • Loss function: CrossEntropyLoss
  • Optimizer: Adam (learning rate 1e-4)
  • Batch size: 16
  • Epochs: 5

Implementation Environment

Trained in Google Colab using GPU acceleration. The dataset was stored on Google Drive for easy access and reproducibility.
